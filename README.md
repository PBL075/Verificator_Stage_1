Verificator_stage_1 
Un program în C# care calculează performanțele unui Audi RS3 după aplicarea unei modificări de tip Stage 1.

Ce face programul:
Definire mașină: Stochează datele de bază ale unui RS3 (425 CP stock și accelerație de 3.8 secunde).

Calcul Stage 1: Adaugă o putere suplimentară de 50 CP la valoarea inițială a motorului.

Afișare rezultate: Compară puterea stock cu cea finală și afișează specificațiile complete (nume, putere totală și timpul de razgon).

Detalii tehnice (Tipuri de date folosite):
string: Pentru modelul mașinii ("RS3").

int: Pentru valorile întregi ale cailor putere (425 CP, 50 CP).

double: Pentru precizie la timpul de accelerație (3.8 sec).

Concatenare: Combinarea variabilelor cu text pentru a genera un mesaj clar în consolă.

Cum se rulează:
Deschide proiectul în Visual Studio sau VS Code.

Rulează comanda dotnet run.

Vei vedea mesajul: "RS3 stage 1 [PutereTotală] HP cu razgon de 3.8 sec".
